# Project: Packet Tracer — Connect Two Computers

Guide section: https://beej.us/guide/bgnet0/html/

## Goal
Use Cisco Packet Tracer to simulate a direct connection between two PCs,
assign IP addresses, and verify connectivity with ping.

## Prerequisites
Install Packet Tracer — see `appendices/packet_tracer_install/README.md`.

## Steps
1. Open Packet Tracer and create a new project.
2. Drag two PC icons onto the canvas.
3. Connect them with a crossover (or auto) cable.
4. Assign static IP addresses on the same /24 subnet.
5. Use `ping` from PC0's CLI to reach PC1.

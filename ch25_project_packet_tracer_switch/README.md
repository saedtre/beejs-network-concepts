# Project: Packet Tracer — Using a Switch

Guide section: https://beej.us/guide/bgnet0/html/

## Goal
Add a Layer 2 switch to a Packet Tracer topology to connect multiple PCs
on the same LAN and observe how switching differs from a hub.

## Steps
1. Create a topology with 3 PCs and one switch.
2. Connect each PC to the switch with straight-through cables.
3. Assign IPs on the same /24 subnet.
4. Send pings between all pairs of PCs.
5. Observe the MAC address table on the switch.

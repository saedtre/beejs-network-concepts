# Project: Computing and Finding Subnets

Guide section: https://beej.us/guide/bgnet0/html/

## Goal
Implement helper functions to perform subnet arithmetic:
- Given an IP and prefix length, find the network address and broadcast address.
- Determine whether two IP addresses are on the same subnet.
- List all host addresses in a subnet.

## Run
```bash
python subnet.py 192.168.1.42 24
```

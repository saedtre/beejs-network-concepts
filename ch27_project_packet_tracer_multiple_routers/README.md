# Project: Packet Tracer — Multiple Routers

Guide section: https://beej.us/guide/bgnet0/html/

## Goal
Build a Packet Tracer topology with multiple routers, configure static
routes, and verify end-to-end connectivity across the multi-hop network.

## Steps
1. Build a topology with at least 3 routers and 3 distinct subnets.
2. Connect routers with serial or Ethernet links (point-to-point subnets).
3. Assign IP addresses to all interfaces.
4. Add static routes on each router so every subnet can reach every other.
5. Trace a ping from one end of the network to the other.

# Project: Packet Tracer — Using a Router

Guide section: https://beej.us/guide/bgnet0/html/

## Goal
Introduce a router to connect two separate /24 subnets in Packet Tracer
and verify cross-subnet communication.

## Steps
1. Create two LANs (each with a switch and 2 PCs) on different subnets.
2. Connect both switches to a router (one interface per LAN).
3. Configure router interface IPs as default gateways for each subnet.
4. Set default gateways on all PCs.
5. Ping across subnets to verify routing.

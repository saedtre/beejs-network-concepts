# Appendix: Installing Cisco Packet Tracer

Guide section: https://beej.us/guide/bgnet0/html/

## Steps
1. Create a free Cisco Networking Academy account at https://www.netacad.com/
2. Download Packet Tracer for your OS from the Skills for All portal.
3. Install and log in with your Networking Academy credentials.
